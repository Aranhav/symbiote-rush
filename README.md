# Symbiote Rush - Spider-Verse Edition

An action-packed endless runner game featuring Venom with authentic Spider-Man: Into the Spider-Verse comic book aesthetics.

## Play Now

[Play Symbiote Rush](https://symbiote-rush.up.railway.app/)

## Features

### Gameplay
- Play as Venom running through NYC streets
- Dodge urban obstacles: taxi cabs, fire hydrants, dumpsters, construction barriers
- Avoid Green Goblin's pumpkin bombs
- Escape Electro's lightning strikes from storm clouds
- Double jump mechanics
- Progressive difficulty

### Spider-Verse Aesthetics
- Ben-Day halftone dot patterns
- CMYK color misregistration
- Chromatic aberration effects
- Comic panel borders with accent colors
- Speed lines and afterimages
- Ink splatters on impacts
- Bold onomatopoeia (POW!, CRASH!, ZZZAP!)
- Glitch/distortion effects
- Scan lines and vignette

### Characters
- **Venom** - Main playable character with tendrils, tongue, and spider symbol
- **Green Goblin** - Flying enemy on glider throwing pumpkin bombs
- **Electro** - Lightning bolts striking from storm clouds

## Controls

- **Space / Up Arrow / W** - Jump (press again for double jump)
- **Down Arrow / S** - Duck / Fast fall
- **Click / Tap** - Jump (mobile)

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- Web Audio API
- Express.js (for deployment)

## Local Development

```bash
# Clone the repository
git clone https://github.com/yourusername/symbiote-rush.git

# Navigate to project
cd symbiote-rush

# Install dependencies
npm install

# Start the server
npm start

# Open http://localhost:3000
```
